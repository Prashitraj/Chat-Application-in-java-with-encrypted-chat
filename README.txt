To run server application:

javac Server.java 
java Server <mode>

To run client application:
javac Client.java
java Client <mode>

mode can be 1,2 or 3 depending on whether we want to run step 1,2 or 3 respectively.

To send message:
@<username> <message>

Message should be single line and of length less than 53 bytes

To unregister:
unregister
